# ⚽ EntrenaFútbol - App de Entrenamiento de Fútbol

**EntrenaFútbol** es una aplicación móvil multiplataforma diseñada para jugadores y aficionados al fútbol que desean mejorar su rendimiento, organizar rutinas de entrenamiento y llevar un registro de su progreso. Compatible con **Android** y **iOS**.

---

## 📱 Plataformas Compatibles

- ✅ Android (versión 8.0 en adelante)
- ✅ iOS (versión 13.0 en adelante)

---

## 🚀 Características Principales

- 🏃‍♂️ Ejercicios personalizados 
- 📋 Planes de entrenamiento semanales
- 📈 Seguimiento de rendimiento y estadísticas personales
- 🧠 Consejos tácticos y mentales
- 🔔 Recordatorios y notificaciones de sesiones
- 🌐 Sincronización en la nube y uso sin conexión

---

## 🛠️ Tecnologías Utilizadas

- **Framework:** [Flutter](https://flutter.dev/) (Dart)
- **Backend:** Firebase / Node.js (opcional)
- **Base de datos:** Firestore o SQLite (modo local)
- **Autenticación:** Firebase Auth / Apple Sign-In / Google Sign-In
- **Notificaciones:** Firebase Cloud Messaging / Local Notifications

---

## 📦 Instalación y Ejecución

### Requisitos previos

- Tener instalado [Flutter](https://docs.flutter.dev/get-started/install)
- Tener configurado Android Studio y/o Xcode (para pruebas en Android o iOS)

### Pasos para correr la app:

```bash
# Clona el repositorio
git clone https://github.com/tu_usuario/entrenafutbol-app.git
cd entrenafutbol-app

# Instala las dependencias
flutter pub get

# Corre la app en un emulador o dispositivo físico
flutter run
